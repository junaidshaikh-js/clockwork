## Features

- Add a task
- Edit/Delete a task
- Pomodoro Feature
- Start/Pause/Reset Clock
- Dark Mode
- The data of tasks is saved even after refreshing the page on local storage.
- The page title time will continuously change and sync with the current Pomodoro timer.
